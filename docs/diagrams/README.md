# Diagrams

Mermaid diagram sources and rendered images live here.

- Edit `.mmd` source files.
- Render PNG outputs with `bun run diagrams`.
- Embed PNGs from docs with relative Markdown image links.
- Keep source and output names paired, for example `backend-configuration.mmd` and `backend-configuration.png`.

Use high-resolution PNGs for stable GitHub rendering and shareable docs. Do not paste large Mermaid blocks into docs when the diagram belongs here.
